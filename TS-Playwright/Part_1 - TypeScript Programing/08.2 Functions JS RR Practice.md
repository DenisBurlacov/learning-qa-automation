### Задание 6.1 — Первый callback

Напиши:

- функцию `executeOperation` — принимает два числа и операцию (функцию двух чисел возвращающую число), возвращает число
- она принимает два числа и **функцию-операцию** как callback
- выполняет операцию и возвращает результат

```typescript
const add = (x: number, y: number) => x + y;

executeOperation(10, 5, multiply); // 50
executeOperation(10, 5, subtract); // 5

// Можно передать inline arrow function:
executeOperation(10, 5, (x, y) => x ** y); // 100000
```
