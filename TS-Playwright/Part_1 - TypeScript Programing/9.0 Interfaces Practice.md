## Как работать с этим файлом

Создай папку `practice/interfaces/` в своём проекте. Для каждого блока — отдельный файл. Запускай через `tsx имяФайла.ts`.

Алгоритм:

1. Прочитай условие
2. Попробуй написать решение сам
3. Запусти — убедись что TypeScript не ругается
4. Сравни со своим решением в разделе РЕШЕНИЯ

**Главная цель практики Interface** — научиться думать контрактами: "что должен уметь этот объект" ещё до написания реализации.

2. `IComponent` — Interface для переиспользуемых компонентов (header, modal, datepicker)
3. `ITestData` — Interface для фабрики тестовых данных

Для каждого Interface создай по одному классу-реализации. Напиши три теста которые используют все созданные Interface через переменные с типом Interface (не конкретного класса).

**4.1**

```typescript
interface Shape {
    color: string;
    area(): number;
    perimeter(): number;
    describe(): string;
}

class Circle implements Shape {
    constructor(public color: string, private radius: number) {}
    area(): number { return Math.PI * this.radius * this.radius; }
    perimeter(): number { return 2 * Math.PI * this.radius; }
    describe(): string {
        return `${this.color} Circle — area: ${this.area().toFixed(2)}, perimeter: ${this.perimeter().toFixed(2)}`;
    }
}

class Rectangle implements Shape {
    constructor(public color: string, private width: number, private height: number) {}
    area(): number { return this.width * this.height; }
    perimeter(): number { return 2 * (this.width + this.height); }
    describe(): string {
        return `${this.color} Rectangle — area: ${this.area()}, perimeter: ${this.perimeter()}`;
    }
}

function printShapeInfo(shape: Shape): void {
    console.log(shape.describe());
}

printShapeInfo(new Circle("red", 5));
printShapeInfo(new Rectangle("blue", 4, 6));
```

**4.2**

```typescript
interface DataRepository<T extends Entity> {
    getById(id: number): Promise<T | null>;
    getAll(): Promise<T[]>;
    create(data: Omit<T, "id" | "createdAt" | "updatedAt">): Promise<T>;
    delete(id: number): Promise<boolean>;
}

class InMemoryUserRepository implements DataRepository<User> {
    private users: User[] = [];

    async getById(id: number): Promise<User | null> {
        return this.users.find(u => u.id === id) ?? null;
    }

    async getAll(): Promise<User[]> {
        return [...this.users];
    }

    async create(data: Omit<User, "id" | "createdAt" | "updatedAt">): Promise<User> {
        const user: User = {
            id: Date.now(),
            createdAt: new Date(),
            updatedAt: new Date(),
            ...data
        };
        this.users.push(user);
        return user;
    }

    async delete(id: number): Promise<boolean> {
        const index = this.users.findIndex(u => u.id === id);
        if (index === -1) return false;
        this.users.splice(index, 1);
        return true;
    }
}
```
