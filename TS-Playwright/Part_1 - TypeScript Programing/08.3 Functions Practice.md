## Как работать с этим файлом

Создай папку `practice/functions/` в своём проекте. Для каждого блока — отдельный файл. Запускай через `tsx имяФайла.ts`.

Алгоритм работы с каждым заданием:

1. Прочитай условие
2. Попробуй написать решение сам — без подсказок
3. Запусти, проверь вывод
4. Если застрял — посмотри подсказку
5. Сравни со своим решением в разделе РЕШЕНИЯ

**Не смотри решения сразу** — это самое важное правило. Даже неправильная попытка даёт больше, чем чтение готового ответа.

### Задание 3.1 — Перепиши как arrow

Перепиши каждую из следующих named functions как arrow function. Там где возможно — используй implicit return (без фигурных скобок и без `return`).

```typescript
// A — перепиши как arrow с implicit return
function double(n: number): number {
    return n * 2;
}

// B — перепиши как arrow с implicit return
function isEven(n: number): boolean {
    return n % 2 === 0;
}

// C — перепиши как arrow с implicit return
function fullName(first: string, last: string): string {
    return `${first} ${last}`;
}

// D — перепиши как arrow БЕЗ implicit return (здесь несколько строк)
function describeUser(name: string, age: number): string {
    const status = age >= 18 ? "adult" : "minor";
    return `${name} is ${age} years old and is a ${status}`;
}
```

После переписывания — вызови каждую функцию и убедись что результат тот же.

### Задание 3.4 — Функция возвращает функцию

Напиши arrow function `makeMultiplier` которая:

- принимает число `factor`
- **возвращает новую arrow function**, которая принимает число и умножает его на `factor`

Это называется closure (замыкание) — важная концепция JS/TS.

```typescript
(function() {
    // код выполнится сразу
})();
```

В TypeScript функции можно хранить как свойства объекта (методы объекта через функциональное присваивание):

```typescript
const calculator = {
    add: function(a: number, b: number): number { return a + b; },
    // ... твои функции
};
```

Создай объект `calculator` с четырьмя методами: `add`, `subtract`, `multiply`, `divide`. Для `divide` добавь проверку — если делитель равен 0, возвращай строку `"Error: division by zero"`.

```typescript
const retryAction = async (
    action: () => Promise<void>,
    maxRetries: number = 3,
    delayMs: number = 1000
): Promise<void> => {
    let lastError: Error;
    for (let i = 0; i < maxRetries; i++) {
        try {
            await action();
            return;
        } catch (error) {
            lastError = error as Error;
            if (i < maxRetries - 1) {
                await new Promise(resolve => setTimeout(resolve, delayMs));
            }
        }
    }
    throw lastError!;
};
```

```typescript
async function verify(
    page: Page,
    type: string,
    selectorOrExpected: string,
    expected?: string
): Promise<void> {
    if (type === "title") await expect(page).toHaveTitle(selectorOrExpected);
    else if (type === "url") await expect(page).toHaveURL(selectorOrExpected);
    else if (type === "visible") await expect(page.locator(selectorOrExpected)).toBeVisible();
    else if (type === "text" && expected !== undefined) {
        await expect(page.locator(selectorOrExpected)).toHaveText(expected);
    }
}
```
