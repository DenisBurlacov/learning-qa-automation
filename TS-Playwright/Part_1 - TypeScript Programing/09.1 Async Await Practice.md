## Как работать с этим файлом

Создай папку `practice/async/` в своём проекте. Для каждого блока — отдельный файл. Блоки 1-3 запускай через `tsx имяФайла.ts`. Блок 4 требует Playwright проект.

Алгоритм работы с каждым заданием:

1. Прочитай условие
2. Попробуй написать решение сам — без подсказок
3. Запусти, проверь вывод
4. Если застрял — посмотри подсказку
5. Сравни со своим решением в разделе РЕШЕНИЯ

**Не смотри решения сразу** — это самое важное правило.

### Задание 3.4 — Таймаут для Promise

Напиши функцию `withTimeout<T>(promise: Promise<T>, ms: number): Promise<T>` которая:

- принимает любой Promise и таймаут в миллисекундах
- если Promise не завершается за `ms` миллисекунд — выбрасывает `new Error("Timeout after [ms]ms")`
- если Promise завершается вовремя — возвращает его результат

Проверь:

```typescript

// "done"

// Медленный запрос — таймаут
await withTimeout(delay(1000), 300);
// Error: Timeout after 300ms
```

const waitForElement = async (
    page: Page,
    selector: string,
    maxAttempts: number = 5,
    intervalMs: number = 500
): Promise<boolean> => {
    for (let i = 0; i < maxAttempts; i++) {
        const element = page.locator(selector);
        const count = await element.count();
        if (count > 0) return true;
        await page.waitForTimeout(intervalMs);
    }
    throw new Error(`Element ${selector} not found after ${maxAttempts} attempts`);
};

    add(task: () => Promise<void>): void {
        this.tasks.push(task);

**5.2**

```typescript
function memoizeAsync<T>(fn: (key: string) => Promise<T>): (key: string) => Promise<T> {
    const cache = new Map<string, T>();
    return async (key: string): Promise<T> => {
        if (cache.has(key)) {
            console.log(`[CACHE HIT] ${key}`);
            return cache.get(key)!;
        }
        console.log(`[FETCH] ${key}`);
        const result = await fn(key);
        cache.set(key, result);
        return result;
    };
}
```
